# Multi-shop e-commerse API. 
This project built with NestJS, TypeORM and PostgreSQL

# Technologies

- NestJS
- TypeORM
- PostgreSQL
- JWT Authentication
- Swagger

# Running the app

'''bash
npm run start:dev
'''

# APi Documentation

Swagger: `http://localhost:3000/api`

# Roles

User: Enables browsing shops and products, and submitting shop opening applications.
Shop: Provides management tools for shops and products.
Admin: Handles shop moderation and platform administration.
